Operations

Deployment Commands

Common azd and Docker commands for managing the accelerator deployment.

Full Deployment

# Provision infrastructure + deploy all services

azd up


# Deploy all services to existing infrastructure

azd deploy


# Deploy via helper script

./scripts/azd-deploy.sh all

Selective Deployment

# Deploy a specific accelerator backend

azd deploy accel-001


# Deploy a specific accelerator frontend

azd deploy accel-001-fe


# Deploy specific accelerator via script

./scripts/azd-deploy.sh 001

Local Development

# Run all accelerators locally

docker-compose -f docker-compose.accelerators.yml up


# Run a single accelerator

docker-compose -f docker-compose.accelerators.yml up accel-001-backend


# Rebuild and run

docker-compose -f docker-compose.accelerators.yml up --build

Diagnostics

# Check environment

azd env get-values


# Show deployed services

azd show


# Run live e2e tests

npx playwright test --config playwright.live.config.ts
