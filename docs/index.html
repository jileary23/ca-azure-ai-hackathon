Compliance

California Regulatory Alignment

Every accelerator is designed to comply with California's AI governance framework from day one.

EO

EO N-12-23

GenAI guidelines for California state agencies. Defines responsible use, risk assessment, and procurement standards for AI systems.

EO

EO N-5-26

AI procurement and vendor attestation requirements. Mandates transparency scoring and bias audits for all AI vendor contracts.

SB

SB 53

AI safety legislation. Establishes safety requirements, testing mandates, and reporting obligations for high-risk AI deployments.

§

CCPA / CPRA

California consumer privacy protections. All accelerators enforce PII filtering, data minimization, and consent-aware processing.

26

Envision 2026

California's digital government strategy. Accelerators align with the vision of modernized, accessible, and AI-augmented public services.

BP

Breakthrough Project

Permitting modernization initiative. Accelerator 004 directly supports the statewide effort to streamline permit processing.

Quick Start

Up and Running in 60 Seconds

No Azure credentials required — mock mode runs everything locally.

terminal 
# Clone the repository
$ git clone https://github.com/msftsean/ca-hackathon
$ cd ca-hackathon

# Install dependencies
$ npm run setup

# Start in mock mode — no Azure needed
$ npm start

✓ Platform running at http://localhost:3000
✓ 8 accelerators ready (mock mode)
