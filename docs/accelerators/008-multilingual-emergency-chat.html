Multilingual Emergency Chatbot

Real-time emergency alerts, shelter locations, and air quality data in 15+ languages — built for California's most vulnerable communities.

1 Problem Statement

During the 2025 LA fires, non-English speakers couldn't access critical evacuation orders, shelter information, or air quality warnings. California responded by launching a 70-language wildfire chatbot — demonstrating the urgent need for multilingual emergency communication. This accelerator is the open-source blueprint for that capability: making emergency information accessible to every Californian regardless of language, in low-bandwidth and SMS-ready formats.

2 Solution Overview

Emergency Alerts by ZIP

Real-time alerts for fire, earthquake, flood, and tsunami events filtered by ZIP code with severity levels and actionable instructions.

Shelter Locator

Nearby shelter search with capacity, ADA accessibility, pet-friendliness, and available services — filtered by radius from any ZIP code.

Air Quality Monitoring

AQI readings with health guidance levels, especially critical during wildfire season for vulnerable populations.

Real-Time Translation

15+ language support with instant translation, designed for low-bandwidth connections and SMS delivery during infrastructure outages.

3 Architecture — 3-Agent Pipeline

Q

QueryAgent

Detects language, extracts location (ZIP code), identifies emergency type (fire/earthquake/flood/tsunami), and normalizes queries for low-bandwidth input formats.

R

RouterAgent

Routes to alert feeds, shelter database, air quality API, or translation service. Prioritizes life-safety queries and handles SMS-format routing.

A

ActionAgent

Retrieves geo-filtered alerts, locates accessible shelters, fetches AQI data, and translates all responses into the user's preferred language.

4 API Endpoints

Method Path Description
GET /health Health check
POST /api/chat Main chat interface
GET /api/status Service status
GET /api/alerts?zip=&lang= Emergency alerts by ZIP
GET /api/shelters?zip=&radius= Shelter locator
GET /api/air-quality?zip= Air quality index
POST /api/translate Text translation

5 Data Models

Alert

Emergency alert with type, severity, and actionable instructions.

{
  "alert_id": "string",
  "type": "fire | earthquake | flood | tsunami",
  "severity": "watch | warning | emergency",
  "title": "string",
  "description": "string",
  "zip_codes": ["string"],
  "issued_at": "ISO 8601",
  "expires_at": "ISO 8601"
}

Shelter

Shelter with capacity, accessibility, and services info.

{
  "shelter_id": "string",
  "name": "string",
  "address": "string",
  "capacity": 200,
  "current_occupancy": 85,
  "ada_accessible": true,
  "pets_allowed": true,
  "services": ["food", "medical", "wifi"]
}

AirQualityReading

AQI reading with health guidance for sensitive groups.

{
  "zip": "string",
  "aqi": 142,
  "category": "Unhealthy for Sensitive Groups",
  "dominant_pollutant": "PM2.5",
  "health_guidance": "string",
  "measured_at": "ISO 8601"
}

TranslateRequest / Response

Translation payload for 15+ supported languages.

// Request
{
  "text": "string",
  "source_lang": "en",
  "target_lang": "es | zh | vi | ko | tl | ..."
}
// Response
{
  "translated_text": "string",
  "detected_lang": "en",
  "confidence": 0.98
}
