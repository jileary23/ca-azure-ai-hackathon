ACCEL-007 EDD UI · DI · PFL

EDD Claims Assistant

AI-powered claims assistance for Unemployment Insurance, Disability Insurance, and Paid Family Leave with real-time benefit calculations.

1 Problem Statement

California's Employment Development Department (EDD) is the state's most-criticized agency. Unemployment Insurance (UI), Disability Insurance (DI), and Paid Family Leave (PFL) claims face persistent processing delays. Phone wait times regularly exceed 60 minutes, and confusing eligibility rules frustrate claimants who are already in financial distress. The system needs intelligent self-service tools that can answer questions instantly and accurately.

2 Solution Overview

Benefit Calculator

Real-time UI/DI/PFL benefit calculations using 2024 California rates with min/max caps and quarterly earnings inputs.

Claim Status Lookup

Instant claim status retrieval with processing stage tracking and estimated completion timelines.

Eligibility Screening

Pre-screening tool that evaluates eligibility criteria before formal application, reducing rejected claims.

Document Checklists

Personalized document requirement lists based on claim type, ensuring complete submissions from the start.

Processing Timelines

Transparent timeline visibility for each claim type so claimants can set realistic expectations.

Smart Escalation

Confidence-based escalation to live agents with full context transfer — no repeating information.

3 Architecture — 3-Agent Pipeline

Q

QueryAgent

Detects claim type (UI/DI/PFL), extracts earnings data, identifies claimant intent (status check, calculator, eligibility), and filters PII from conversations.

R

RouterAgent

Routes to benefit calculator, claim status, eligibility checker, or document checklist. Triggers escalation when confidence drops below threshold.

A

ActionAgent

Calculates benefits with 2024 CA rates, retrieves claim statuses, generates document checklists, and creates escalation tickets with full conversation context.

4 API Endpoints

Method Path Description
GET /health Health check
POST /api/chat Main chat interface
GET /api/status Service status
POST /api/claim-status Claim status lookup
POST /api/eligibility Eligibility check
POST /api/document-checklist Required documents
POST /api/benefits/calculate UI/DI/PFL benefit calculator
GET /api/claims/{type}/timeline Processing timeline
GET /api/claims/{type}/requirements Claim requirements
POST /api/escalate Live agent escalation

5 Data Models

BenefitCalculationRequest

Input for benefit calculations with quarterly earnings.

{
  "claim_type": "ui | di | pfl",
  "quarterly_earnings": [0.00, 0.00, 0.00, 0.00],
  "base_period_start": "YYYY-MM-DD",
  "base_period_end": "YYYY-MM-DD"
}

BenefitCalculationResponse

Calculated benefit amounts with CA rate caps.

{
  "weekly_benefit": 450.00,
  "max_weekly": 450.00,
  "min_weekly": 50.00,
  "total_benefit": 11700.00,
  "duration_weeks": 26,
  "highest_quarter": 7500.00
}

ClaimStatus / ClaimTimeline

Current claim state and processing milestones.

{
  "claim_id": "string",
  "type": "ui | di | pfl",
  "status": "filed | under_review | approved
    | denied | appealed",
  "timeline": [
    { "stage": "string", "days": 0 }
  ]
}

EscalationResponse

Live agent handoff with full context transfer.

{
  "ticket_id": "string",
  "queue_position": 12,
  "estimated_wait_minutes": 15,
  "context_transferred": true,
  "reason": "string",
  "priority": "normal | high | urgent"
}
